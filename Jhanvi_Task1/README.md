# NeuroFlow AI: 
- A modern and responsive AI Productivity Tool Landing Page built using HTML5 and CSS3 during my Web Development Internship at Oasis Infobyte. This project focuses on creating a clean, modern, and visually engaging user interface with smooth animations, interactive sections, and responsive design.

## Project Highlights:
- Modern dark-themed landing page UI
- Fully responsive design for all screen sizes
- Smooth scrolling navigation system
- Interactive hover effects and transitions
- Clean and organized code structure
- Professional gradient buttons and modern styling
- Mobile-friendly responsive layout
- Modern AI productivity website concept

## Features:
- Responsive Navigation Bar
- Hero Section with Modern UI
- AI Productivity Features Section
- Statistics Section
- Pricing Plans Section
- Testimonials Section
- Interactive Hover Animations
- Smooth Scroll Navigation
- Responsive Mobile Layout
- Clean UI/UX Design

## Tech Stack:
- HTML5
- CSS3
- Google Fonts
- Font Awesome
- Flexbox
- Responsive Web Design
