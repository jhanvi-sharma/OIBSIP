# Personal Portfolio Website:
A modern and responsive Personal Portfolio Website built using HTML5 and CSS3 during my Frontend Development Internship at Oasis Infobyte. This project showcases my skills, projects, learning journey, and contact information through a clean and professional user interface.

## Project Highlights:
* Modern dark-themed portfolio design
* Fully responsive layout for all screen sizes
* Smooth hover effects and transitions
* Professional hero section with introduction
* Skills showcase with icons
* Project cards with live demo links
* Contact section with clickable social links
* Beginner-friendly and organized code structure
* Clean UI/UX inspired by modern developer portfolios


## Features:
* Responsive Navigation Bar
* Hero Section with Introduction
* About Me Section
* Skills Section with Icons
* Project Showcase Section
* Learning Journey Section
* Contact Information Section
* Interactive Hover Effects
* Responsive Mobile-Friendly Layout
* Social Media Integration
